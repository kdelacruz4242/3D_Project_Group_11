# Description:

This project is a chaotic, rage-inducing obstacle course designed to
challenge the player’s expectations and reactions. The core idea is to
create a game where the player cannot fully trust what they see or how
the game behaves. Players will navigate through unpredictable
environments filled with traps and deceptive mechanics. These may
include fake floors, disappearing platforms, and misleading visual cues
that create uncertainty and tension. The game may also experiment with
altered controls, such as reversed or inconsistent movement, forcing
players to adapt quickly. The overall experience is meant to feel
intentionally unfair but engaging, encouraging trial-and-error gameplay
and persistence.
